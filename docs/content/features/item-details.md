# Item details

Tap any item in a list to open its detail screen where you can edit name, quantity, and category.

## Editable fields

| Field | Description |
|-------|-------------|
| **Name** | Display name shown in the list |
| **Quantity** | Optional count or weight (g, kg, ml, L) |
| **Category** | Which category header the item appears under |

Changes save automatically when you navigate back.

## Catalog items vs custom items

- **Built-in catalog items** — changing the category updates your personal preference. The item stays in the catalog with your chosen category.
- **Custom items** — fully editable. You can also remove them from your catalog memory.

## Removing custom items

For items you added yourself, a **Remove from catalog** option is available. This removes the item from your personal catalog so it won't appear in autocomplete again. It does not remove the item from the current list until you delete it separately.

## Quantities

Quantities support:

- **Count** — whole number (default)
- **Weight** — grams (g) or kilograms (kg)
- **Volume** — millilitres (ml) or litres (L)

Leave quantity empty if you don't need it — the item displays as a simple line entry.
