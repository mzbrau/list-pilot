# Settings & export

Open **Settings** from the gear icon on the main lists overview screen.

## Appearance

Choose how List Pilot looks:

| Option | Behavior |
|--------|----------|
| **System** | Follow your device's light/dark setting |
| **Light** | Always use light theme |
| **Dark** | Always use dark theme |

Your choice is saved locally and persists across app restarts.

## Export custom catalog

**Export custom catalog** saves your personal catalog changes to a JSON file:

- Custom items you added
- Built-in items you recategorized

### On Android

The file is saved to your public **Downloads** folder as `list-pilot-catalog-export-YYYY-MM-DD.json`. Open your Files app → Downloads to find it.

### On other platforms

The file is saved to your Downloads folder or app documents directory.

:::tip Before uninstalling
Always export your catalog before uninstalling or switching devices.
:::

## App version

The settings sheet shows the current app version at the bottom, useful when reporting issues or verifying you've installed the latest release.
