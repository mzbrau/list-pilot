# Install on Android

List Pilot is distributed as an APK through [GitHub Releases](https://github.com/mzbrau/shop-flow/releases). Google Play distribution is planned for the future.

## Download

1. Open the [GitHub Releases](https://github.com/mzbrau/shop-flow/releases) page.
2. Download the latest `list-pilot-X.Y.Z.apk` file.
3. On your phone, allow installation from unknown sources for your browser or file manager (if prompted).
4. Open the downloaded APK and install.

:::tip Upgrading
If you already have List Pilot installed with the same signing key, installing a newer release upgrades in place and **preserves all your data**. See [Upgrading](/docs/upgrading) for migration notes.
:::

## Requirements

- Android device (API level supported by Flutter stable)
- No Google account required
- No internet connection required after install

## Google Play

List Pilot is not yet available on Google Play. Check back on the [documentation site](https://mzbrau.github.io/shop-flow/) for updates.

## Migrating from Shop Flow

If you previously installed **Shop Flow** (the old app name and package ID), you cannot upgrade in place. Follow the [Upgrading](/docs/upgrading) guide to export your catalog and migrate to List Pilot.
